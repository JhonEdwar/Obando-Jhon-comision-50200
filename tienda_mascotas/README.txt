# Aplicación 1 
    AppTienda: que permite administrar una tienda enfocada en productos para mascotas,
    los datos de los animales tipo perro o gato 
    y los diferentes productos para estos tipos de mascotas.

- Los modelos CRUD que presenta esta aplicacion son:
  - Mascotas
  - Productos
  - Medicamentos

ingreso a la app:
localhost:8000/apptienda/

_____________________________________

# Aplicación 2:
    AppClientes: 
    el sistema se enfoca en guardar y administrar los datos de un cliente

- Los modelos CRUD que presenta esta aplicacion son:
  - Clientes

  ingreso a la app:
localhost:8000/apptienda/
___________________________________________
Consulta de información a travez del menú o los siguientes enlaces:

1. Conslutar lista de clientes: localhost /appclientes/clientes/
2. Conslutar lista de mascotas: localhost /apptienda/mascotas/
3. Conslutar lista de productos: localhost /apptienda/productos/
4. Conslutar lista de productos: localhost /apptienda/medicamentos/

__________________________________________
Formularios de creación:

1. Crear clientes: localhost /appclientes/clientes_form/
2. Crear mascotas: localhost /apptienda/mascotas_form/
3. Crear productos: localhost /apptienda/productos_form/
4. Crear productos: localhost /apptienda/medicamentos_form/

__________________________________________
Campo de búsqueda de productos:

1.Busqueda del nombre del producto por un patron en el nombre: localhost /apptienda/buscar/

__________________________________________
Perfil de usuario 

1. registro: localhost /apptienda/register/
2. login: localhost /apptienda/login/
3. logout: localhost /apptienda/logout/
4. editar perfil: localhost /apptienda/editar_perfil/
4. agregar avatar: localhost /apptienda/agregar_avatar/
