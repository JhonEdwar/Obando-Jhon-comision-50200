# Aplicacion que permite administrar una tienda enfocada en productos para mascotas,
el sistema se enfoca en guardar los datos de un cliente, 
los datos de los animales tipo perro o gato 
y los diferentes productos para estos tipos de mascotas.

- Los modelos CRUD que presenta esta aplicacion son:
  - Clientes
  - Mascotas
  - Productos
  - Medicamentos

ingreso a la app:
localhost:8000/apptienda/
___________________________________________
Consulta de información a travez del menú o los siguientes enlaces:

1. Conslutar lista de clientes: localhost:8000/apptienda/clientes/
2. Conslutar lista de mascotas: localhost:8000/apptienda/mascotas/
3. Conslutar lista de productos: localhost:8000/apptienda/productos/
4. Conslutar lista de productos: localhost:8000/apptienda/medicamentos/

__________________________________________
Formularios de creación:

1. Crear clientes: localhost:8000/apptienda/clientes_form/
2. Crear mascotas: localhost:8000/apptienda/mascotas_form/
3. Crear productos: localhost:8000/apptienda/productos_form/
4. Crear productos: localhost:8000/apptienda/medicamentos_form/

__________________________________________
Campo de búsqueda de productos:

1.Busqueda del nombre del producto por un patron en el nombre: localhost:8000/apptienda/buscar/

__________________________________________
Perfil de usuario 

1. registro: localhost:8000/apptienda/register/
2. login: localhost:8000/apptienda/login/
3. logout: localhost:8000/apptienda/logout/
4. editar perfil: localhost:8000/apptienda/editar_perfil/
4. agregar avatar: localhost:8000/apptienda/agregar_avatar/
